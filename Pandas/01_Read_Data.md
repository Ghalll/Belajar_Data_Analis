# Pandas

Library Pandas digunakan untuk membaca file, bisa file dengan ekstensi `.csv` atau ekstensi `,xlsl`

Contoh code untuk mengimport lilbrary pandas

```bash
import pandas as pd
```
---

### Fungsi Read Data

- `read_csv`
  
  Digunakan untuk membaca file dengan ekstensi ```csv```

- `read_excel`
  
  Digunakan untuk membaca file dengan ekstensi ```xlsl```
