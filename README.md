# Data Analis

Adalah kegiatan untuk mengumpulkan, membersihkan, menganalisis data lalu memvisualisasikan data tersebut

---

Pertama perlu install beberapa library berikut :

1. **Pandas** -> Baca data,bersihin,filter,agregasi

```bash
pip install pandas
```

2. **Numpy** -> Dipakai buay array dan operasi matematika.

```bash
pip install numpy
```

3. **Matplotlib**

```bash
pip install matplotlib
```

4. **Seaborn**

```bash
pip install seaborn
```

5. **Statsmodels**

```bash
pip install statsmodels
```
