# Data Analis

Adalah kegiatan untuk mengumpulkan, membersihkan, menganalisis data lalu memvisualisasikan data tersebut

---

Pertama perlu install beberapa library berikut :

1. **Pandas** (Load Data)

   Untuk baca data, bersihin, filter, agregasi.

   ```bash
   pip install pandas
   ```

2. **Numpy** (Fondasi numerik)

   Dipakai buat array dan operasi matematika cepat.

   ```bash
   pip install numpy
   ```

3. **Matplotlib** (Visualisasi Dasar)
   
   Untuk buat grafik line, bar, histogram, titik-titik dari data.
   
   ```bash
   pip install matplotlib
   ```

4. **Seaborn** (Visual Statistik)

   Grafik lebih rapi dan cepat untuk EDA.
   
   ```bash
   pip install seaborn
   ```

5. **Statsmodels** (Statistik serius)

   Regresi, uji hipotesis, time series.
   
   ```bash
   pip install statsmodels
   ```
