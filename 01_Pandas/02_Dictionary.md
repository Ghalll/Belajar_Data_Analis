# Dictionary

Adalah struktur data yang memiliki 2 unsur yaitu : **keys** dan **values**, yang mana setiap key digunakan untuk mengakses valuenya.

contoh dalam program :

```bash
europe = {
    'spain' : 'madrid',
    'france' : 'paris',
    'germany' : 'berlin',
    'norway' : 'oslo'
}
```

`spain` = adalah key.

`madrid` = adalah value.
